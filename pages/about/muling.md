# 关于慕棱

慕棱，一个爱折腾但并不想被复杂玩意儿折磨死的大学生。

平时会碰很多东西：服务器、Docker、网站、反代、Cloudflare、EdgeOne、NewAPI、OpenClaw、本地模型、各种 AI 工具和游戏相关服务。兴趣点很杂，但大多围绕一个目标：把想象里的系统真的搭出来。

## 他在折腾什么

最近一段时间，慕棱把大量精力放在 AI 网关和个人模型入口上：

- 修改 New API 前端，做成更贴近自己审美的「洛音云枢」；
- 用 New API 管理渠道、模型、分组和令牌；
- 把 OpenClaw 接到自己的模型入口上；
- 研究不同模型服务的协议差异和兼容问题；
- 在各种代理、接口、日志和配置之间来回横跳。

这些东西不一定都适合公开写成教程，但它们确实构成了这段时间的生活背景。

## 他不想做什么

慕棱有能力搭一套可用的 AI 中转或模型网关，也能教别人搭面板、排配置、查链路。

但这不代表他想把兴趣做成生意。

对他来说，折腾系统的乐趣在于理解、掌控和跑通，而不是每天处理用户、售后、退款、炸号、风控和各种乱七八糟的人类问题。

所以这里更像个人记录，而不是商业项目入口。

## 一点审美

慕棱喜欢二次元风格，也喜欢洛天依。项目如果只是能跑但不好看，他大概率还是会想改。

他不喜欢八股文式的表达，也不喜欢把简单事情包装得很高级。能说人话就说人话，能直接解决就别绕弯。
