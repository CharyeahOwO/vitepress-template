# 慕棱到底是个什么生物

慕棱这家伙简直就是个被景德镇陶瓷耽误的赛博包工头。你以为他在学校安安静静捏泥巴？其实这人半夜三点正盯着一行报红的 Nginx 日志狂骂。

他根本没经过正经计算机科班的毒打，那一身技术全靠在互联网大后方疯狂搭违建练出来的野路子。什么 Docker 反代全栈运维甚至跨国 API 账号池，只要他想搞，徒手就能给你搓出来一套。过程优不优雅不好说，但最后反正就是奇迹般地跑通了。

这人手里那点可怜的生活费，估计全进云厂商的口袋和域名续费的坑里了。买衣服？买好吃的？不存在的，给洛天依打榜充信仰或者买个香港节点折腾才是他的正经事。对了，他还是个究极的洛天依单推人，就是那种能在茫茫盲盒里一发入魂捞出百分之零点五七隐藏款的狗屎运选手。

他天天把自己埋在这堆服务器和接口里，真不是为了以后靠这个卖钱开公司。他单纯就是有一种“这破系统居然敢不按我的意思跑我今天非要干碎它”的强迫症。在配置的废墟里自己当一回上帝，把破烂拼成城堡，这就是慕棱最上头的乐趣。
