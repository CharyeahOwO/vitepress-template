# 关于本站的免责声明

欢迎来到这个伪装得极好的文档站。

## 结构说明

如果你看左边的侧边栏，你会觉得：“哇，好专业！有项目介绍，有技术栈，还有更新日志！”

但当你点进去就会发现，这里的“技术栈”可能只是在罗列慕棱踩过的坑，这里的“更新日志”可能只是某天他一怒之下重装系统的案发现场。

## 适用人群

本站**不适合**以下人群：
- 希望在此找到“三步搭建完美模型网关”手把手教程的新手。
- 试图寻找可以直接复制粘贴且不出错的生产环境代码的伸手党。

本站**极其适合**以下人群：
- 同样被各类大厂 API 和反代配置折磨到掉头发的同道中人。
- 想看看一个非科班野生程序员是怎么用一堆开源组件拼凑出自己的 AI 帝国的围观群众。

## 幕后黑手
