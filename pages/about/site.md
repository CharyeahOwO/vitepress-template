# 这里是什么

这里不是 New API 官方文档，也不是一本完整教程。

它更像一个个人文档馆：放一些关于慕棱、项目、折腾记录和站点说明的内容。

## 为什么存在

很多东西散落在聊天记录、GitHub、服务器、配置文件和脑子里。时间久了，不整理就会变成一团雾。

这个站的作用就是把其中一部分捡出来，放成能被再次看见的样子。

## 写什么

可能会写：

- 项目介绍；
- 折腾记录；
- 技术栈说明；
- 站点更新；
- 一些偏个人的说明；
- 不太适合放在博客正文里的碎片。

## 不写什么

暂时不打算写成完整教程，也不打算把所有细节都公开。

有些东西适合自己留着，有些东西适合以后再说。公开文档不是把工具箱倒在桌上。
