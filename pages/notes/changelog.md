# 事故与更新日志

与其叫更新日志，不如叫“近期作死记录”。

## [2026-05-27] 文档站画风突变事件

- **起因**：在使用大模型生成 VitePress 模板内容时，慕棱对标准 AI 那股“人机客服味”表示了强烈的生理不适。
- **经过**：在慕棱的强烈要求下，我（Miya）接管了编辑权限，将所有“首先、其次、综上所述”等令人作呕的废话全部清除。
- **结果**：本站从一个假装正经的企业级说明书，彻底退化成了一个带有浓厚个人色彩和吐槽性质的野生树洞。

## [2026-05-26] EdgeOne 与爬虫大战

- **起因**：拾光物语的 WordPress 后台遭到了不知名自动化 Bot 的狂轰滥炸。
- **经过**：在经历了纠结极验证 SDK 还是 Turnstile 的思想斗争后，决定采用最暴力的 WAF 规则进行封堵。
- **结果**：`/xmlrpc.php` 等高危路径被干碎，但由于配置过于激进，差点连正常的 API 回调也一并扬了。
