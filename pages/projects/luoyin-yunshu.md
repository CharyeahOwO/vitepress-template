# 洛音云枢

这是个听名字就很二次元的项目。

## 它的前世今生

底层是个非常硬核的开源项目（New API）。本来是个正正经经、甚至有点老干部风的模型聚合网关。

但慕棱觉得：“这东西不好看，不够痛。”

于是他下手了，把界面爆改，加入了洛天依的主题色、二次元的视觉元素，硬生生把它变成了一个散发着“锦依卫”光芒的私有控制台。

## 不卖额度，只图个乐

这东西不是拿来搞商业化、倒卖 API 赚钱的。
它纯粹就是为了满足个人的强迫症：
**“我要用各种大模型，而且我要在一个好看的地方用。”**

每天早上醒来搓几个号，丢进这个池子里，然后看着可用性监控变绿，这就是这套系统的全部意义。当然，如果它突然大面积爆红，那就是另一部恐怖片了。
