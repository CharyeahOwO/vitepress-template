# 洛音云枢

「洛音云枢」是慕棱基于 New API 改造出的个人模型网关前端。

它的核心不是“开一个站卖接口”，而是给自己的 AI 使用环境做一个更顺手、更好看、更可控的入口。

## 名字

「洛音」来自洛天依相关的审美和情绪连接。

「云枢」则更像一个枢纽：把不同模型、渠道、令牌和客户端连接到一起。

合起来，它不是一个冷冰冰的控制台名字，而是更像慕棱自己的东西。

## 它背后是什么

底层仍然是 New API：

- 管理模型；
- 管理渠道；
- 管理分组；
- 管理令牌；
- 统一 OpenAI 兼容入口；
- 给 OpenClaw、酒馆、开发工具和其他客户端提供模型访问层。

但前端视觉、文案和整体气质会更偏个人化。

## 为什么要改

因为默认面板能用，但不一定有归属感。

慕棱不是只想要一个能填 Key 的后台。他更想要一个“这是我的系统”的感觉。

所以洛音云枢里会有更多二次元风格、洛天依气质和个人审美，而不是纯企业后台那种冷淡界面。

## 当前定位

它目前更偏个人使用和实验：

- 统一管理自己的模型入口；
- 给 OpenClaw 等工具供给模型能力；
- 观察不同渠道的可用性；
- 作为个人 AI 基础设施的一部分。

不是商业中转站，也不是公开售卖服务。

## 一句话

洛音云枢是慕棱把 New API 改成“自己会愿意每天打开”的样子。
