# 洛音云枢 (API Gateway)

> 这是一个披着二次元痛车外衣的硬核大模型调度枢纽。

## 严肃的产品定位

在概念上，它是慕棱个人专属的统一模型 API Gateway。

它的核心功能非常正经：
1. **统一入口**：不管后面接的是 OpenAI、Claude 还是 Gemini，前端请求一律只认这一个地址。
2. **渠道分发**：根据模型名称、账号存活状态和额度，动态路由到不同的上游接口。
3. **令牌管理**：给不同的应用（比如 OpenClaw、酒馆）分发独立的 Token，方便追踪是哪个逆子在疯狂消耗额度。

## 不严肃的视觉实现

原版项目叫 New API，是一个非常优秀的开源网关，但它的 UI 充满了那种让慕棱头疼的“企业级管理后台”的冷酷感。

于是，洛音云枢诞生了。

“洛音”代表了他对洛天依的信仰，“云枢”则是它的功能本质。
这套系统被他硬生生注入了大量的二次元元素和特定的主题色。在别人看来这可能花里胡哨，但在他眼里，这是一个“每天必须看一眼才安心”的数字神龛。

## 运维日常

你以为有了网关就一劳永逸了？天真。

它的日常就是：
- 早上 8:00：发现上游几个 OpenAI 号被风控封禁，全部飘红。
- 早上 8:15：骂骂咧咧地手动搓几个新号填进池子里。
- 早上 8:30：看着 Dashboard 上的指示灯全部变绿，心满意足地去上课/捏泥巴。
